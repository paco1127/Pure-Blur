What is PURE BLUR?




PURE BLUR is effective tool to blur the inappropriate contents such as Sexual content, Bloody content and Private information. PURE BLUR aims to prevent minors from exposing inappropriate information and prevent the Streamer or YouTuber work from offending the platform's rules.

The Blur Function of PURE BLUR




You can blur Video,Photo and Audio in PURE BLUR. The blur function is update continously. Below is the inappropriate content what the user can choose to blur:

  • Foul language(Audio)
  • Sexual content(Video, Photo)
  • Bloody content(Photo)
  • New function is coming soon...

The blur effect of PURE BLUR




In PURE BLUR, it has 4 different type of blur effects. The user can select the most suitable effect to blur their videos and images. Below is the blur effect which the user can choose:

  • Normal
  • Blur
  • Mosaic
  • Cover
